# Note Taker

[![GitHub license](https://img.shields.io/badge/License-MIT-blue.svg)](https://atitlan-note-taker.herokuapp.com/)

# Description

The app allows the user to: 
* Add new notes
* Save notes to local storage 
* The user can delete notes
